### Orientações:
  
+ Instalação do Node.(https://nodejs.org/en/download/)
+ Execute no terminal
```
  npm i
```
+ Instalação do prompt
+ Execute no terminal

```
  npm i prompt-sync
```
+ Para executar o projeto basta digitar no terminal
  
```
  node (nome_do_arquivo).js
```
